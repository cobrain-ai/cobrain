# CoBrain Mobile Assets

This directory contains app assets for the CoBrain mobile app.

## Required Assets

Before building for production, add the following image assets:

- `icon.png` - App icon (1024x1024px)
- `adaptive-icon.png` - Android adaptive icon foreground (1024x1024px)
- `splash.png` - Splash screen image (1284x2778px recommended)
- `favicon.png` - Web favicon (48x48px)
- `notification-icon.png` - Android notification icon (96x96px, white/transparent)

## Guidelines

- Use PNG format for all images
- Icons should have no padding (Expo adds padding automatically)
- Splash screen should use the dark background color (#0f172a)
- Notification icon should be single-color (white) with transparency
